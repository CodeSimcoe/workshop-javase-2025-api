2. Les requis

Ce workshop requiert plusieurs outils pour sa mise en œuvre :

  • un IDE Java que vous maitrisez (Eclipse, Intellij IDEA, VS Code, NetBeans, …​)

  • un JDK 24

Certains exercices nécessitent l’exécution de commandes du JDK dans un Shell. Il est donc recommandé de définir la variable système JAVA_HOME et ajouter JAVA_HOME\bin à la variable système PATH.

2.1. Si Eclipse est utilisé

La version minimale requise d’Eclipse est 2025-03 avec le plugin « Java 24 support for Eclipse 2025-03 (4.35) »

Le support Java 24 est disponible à partir de la version 2025-06 d’Eclipse.

2.1.1. Installation du plugin « Java 24 support for Eclipse 2025-03 (4.35) »

Utiliser le menu « Help / Eclipse Marketplace … »

Rechercher « Java 24 »

Cliquer sur « Install »

2.2. Si IntelliJ IDEA est utilisé

Le support Java 24 est disponible à partir de la version 2024.3.4 d’IntelliJ IDEA.
